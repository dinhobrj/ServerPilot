Termos de Serviço

Bem-vindo ao Serverpilot, um bot projetado para melhorar a experiência dos usuários em servidores do Discord, fornecendo ferramentas de administração, automação e interações úteis.

1. Aceitação dos Termos

Ao utilizar o Serverpilot, você concorda com estes Termos de Serviço. Caso não concorde, não utilize o bot.

2. Uso do Bot

  • Você deve ter autorização do proprietário do servidor para adicionar e utilizar o Serverpilot.
  • O bot deve ser utilizado em conformidade com as diretrizes da comunidade do Discord.
  • É proibido utilizar o Serverpilot para fins ilegais, abusivos ou que infrinjam os direitos de terceiros.

3. Modificações

Reservamo-nos o direito de modificar estes Termos de Serviço a qualquer momento, mediante aviso prévio.

4. Responsabilidade

Serverpilot é fornecido "como está". Não nos responsabilizamos por danos diretos ou indiretos causados pelo uso do bot.

5. Contato

Para suporte ou dúvidas, entre em contato através do servidor oficial no Discord: https://discord.gg/ef8fzYtAgP

Política de Privacidade

Esta Política de Privacidade descreve como o Serverpilot coleta, utiliza e protege as informações dos usuários.

1. Coleta de Dados

O Serverpilot pode coletar os seguintes dados, limitados ao necessário para funcionamento:

  • IDs de usuários, servidores, canais e mensagens
  • Configurações feitas pelos administradores do servidor
  • Logs de eventos, quando ativados (ex: entradas/saídas de membros)

2. Uso dos Dados

Os dados coletados são utilizados exclusivamente para fornecer as funcionalidades do bot. Nenhuma informação é vendida, compartilhada ou utilizada para fins publicitários.

3. Armazenamento e Segurança

Os dados são armazenados com segurança em servidores protegidos. Implementamos medidas para evitar acesso não autorizado, alteração ou destruição dos dados.

4. Direitos do Usuário

Você pode solicitar a exclusão de dados relacionados ao seu servidor ou usuário entrando em contato pelo servidor de suporte.

5. Contato

Para solicitações ou dúvidas sobre privacidade, entre em contato pelo servidor oficial: https://discord.gg/ef8fzYtAgP
